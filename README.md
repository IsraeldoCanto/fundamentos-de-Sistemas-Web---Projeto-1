# 🐾 PetShop Amigo Fiel - Sistema Web

## Descrição do Projeto

Este é o projeto da **Fase 1** da disciplina de **Fundamentos de Sistemas Web**. Trata-se de um sistema web para um pet shop, desenvolvido utilizando **somente HTML puro** (sem CSS, Bootstrap ou JavaScript).

## Objetivo

Desenvolver um sistema web front-end para o PetShop Amigo Fiel que permita a visualização de produtos por categorias, com imagens, descrições e valores, além dos serviços oferecidos pelo estabelecimento.

## Metas

1. Criar páginas HTML estruturadas com cabeçalho, corpo e rodapé.
2. Apresentar produtos organizados em 3 categorias, com 2 produtos cada (foto, descrição e valor).
3. Listar os serviços do pet shop com descrição, valor e disponibilidade de tele-busca.
4. Publicar o projeto no GitHub e hospedá-lo via GitHub Pages.

## Estrutura do Site

O site possui **4 páginas** interligadas por navegação:

| Página | Arquivo | Descrição |
|--------|---------|-----------|
| **Início** | `index.html` | Página principal com os produtos separados em 3 categorias |
| **Serviços** | `servicos.html` | Lista de serviços oferecidos pelo pet shop |
| **Sobre Nós** | `sobre.html` | Informações sobre o pet shop, equipe e horário |
| **Contato** | `contato.html` | Informações de contato e formulário de mensagem |

## Categorias de Produtos

- **Alimentação**: Ração premium para cães e gatos
- **Brinquedos**: Bola de borracha e arranhador para gatos
- **Higiene e Cuidados**: Shampoo neutro e escova profissional

## Serviços Oferecidos

- Banho (com tele-busca)
- Tosa (com tele-busca)
- Banho + Tosa (com tele-busca)
- Consulta Veterinária
- Hospedagem (com tele-busca)
- Adestramento

## Funcionalidades das Páginas

### Página Inicial (index.html)
- Cabeçalho com nome do pet shop e menu de navegação
- 3 categorias de produtos (Alimentação, Brinquedos, Higiene)
- 2 produtos por categoria com foto, descrição e valor
- Rodapé com informações de contato e links

### Página de Serviços (servicos.html)
- Tabela com todos os serviços disponíveis
- Cada serviço inclui: descrição, valor e informação de tele-busca (Sim/Não)
- Nota explicativa sobre o serviço de tele-busca

### Página Sobre (sobre.html)
- História, missão e valores do pet shop
- Apresentação da equipe
- Horário de funcionamento

### Página de Contato (contato.html)
- Tabela com todas as formas de contato (telefone, e-mail, WhatsApp, Instagram)
- Formulário para envio de mensagens
- Informações de localização

## Tecnologias Utilizadas

- **HTML5** — Linguagem de marcação para estruturar as páginas
- **Sem CSS** — Nenhuma estilização externa ou interna foi aplicada
- **Sem JavaScript** — Nenhuma funcionalidade dinâmica foi implementada
- **Sem Bootstrap** — Nenhum framework de front-end foi utilizado
